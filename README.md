# Hexo-Tag-Snippets README

## Features

Provide tag snippets for Hexo in Markdown

like 

`-hcb` will turn into 

<pre>
{% codeblock [title] [lang:language] [url] [link text] %}
code snippet
{% endcodeblock %}
</pre>


## Using

`-hcbj`: code block for javascript

`-hquote`: Block Quote
